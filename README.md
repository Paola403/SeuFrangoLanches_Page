# 🍔 Seu Frango Lanches

Landing page moderna e responsiva para uma lanchonete, com foco em conversão de clientes e apresentação dos produtos.

---

## 🚀 Sobre o projeto

O **Seu Frango Lanches** é uma landing page desenvolvida para divulgar uma lanchonete, destacando:

* Cardápio de produtos (hambúrgueres, hot dogs e bebidas)
* Avaliações de clientes
* Localização
* Contato rápido (WhatsApp, Instagram, iFood e telefone)

O objetivo principal é **facilitar o contato com o cliente**.

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3 (com design responsivo)
* JavaScript (interações e funcionalidades)
* Google Fonts

---

## ✨ Funcionalidades

* 📱 Layout totalmente responsivo (mobile-first)
* ⏱️ Status dinâmico (aberto/fechado em tempo real)
* 🎯 Botões de contato direto:

  * WhatsApp
  * Instagram
  * iFood
  * Telefone
* 🎨 Animações suaves (fade-in, hover, microinterações)
* 🧲 Efeito de arrastar no cardápio (drag to scroll)
* 🧠 Destaque automático de seções no menu
* 📍 Integração com Google Maps

---

## 📞 Contato integrado

O projeto possui botões diretos para:

* WhatsApp (pedido rápido)
* Instagram (divulgação)
* iFood (plataforma de delivery)
* Telefone (ligação direta no celular)

---

## 🎯 Objetivo

Criar uma experiência visual atrativa e intuitiva para:

* Aumentar conversões
* Melhorar a experiência do usuário
* Facilitar pedidos
